# Bite of Hope – Food Donation Management System

## Overview

Bite of Hope is a web-based food donation management platform designed to reduce food wastage by connecting food donors with NGOs. The system enables donors to contribute surplus food and allows NGOs to manage and distribute food efficiently to people in need.

## Features

* User Registration and Login
* Role-Based Access (Donor, NGO, Admin)
* Food Donation Management
* NGO Management
* Donation Tracking
* Secure Authentication
* Responsive User Interface
* Admin Monitoring and Management

## Installation

1. Clone the repository.
2. Import the project into Eclipse IDE.
3. Configure Apache Tomcat Server.
4. Create a MySQL database.
5. Update database credentials in the JDBC configuration file.
6. Run the project on Tomcat Server.
